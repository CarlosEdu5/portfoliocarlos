Bem-vindo ao meu portfólio pessoal! Este é um site simples e elegante que apresenta minhas habilidades, projetos e informações de contato como desenvolvedor Front-End. 🌟

Este portfólio foi criado para mostrar meu trabalho e trajetória como estudante de Engenharia de Software, com foco em desenvolvimento web. Aqui você encontrará informações sobre mim, meus projetos práticos e as tecnologias que utilizo. 💻

- **Sobre Mim**: Uma seção dedicada a contar um pouco sobre minha jornada e interesses. 👨‍💻

- **Google Fonts**: Fontes personalizadas (Roboto). 🔤

## 🚀 Como Executar

1. Clone este repositório: `git clone https://github.com/CarlosEdu5/portfolio-carlos-eduardo.git` (substitua pelo link real se houver).
2. Abra o arquivo `principal/index.html` em seu navegador web. 🌐
3. Navegue pelas seções usando os links de navegação. 🧭

Não há dependências externas além de um navegador moderno. Tudo é estático! 😊
